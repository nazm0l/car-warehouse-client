# N&M Automobiles

- Description: This is a car warehouse website. Where user can see how many car in stoke. If user logged in they can add car or restoke item or delivery item etc.

## Live site link: https://car-warehouse-b83ff.web.app/

### Features
- User can create account and login using google
- User can add new car if he/she logged in
- User can delete item
- User can restock item
- User can see particular item which he added

### Technology used
- Firebase auth
- react js
- node js
- mongodb
- firebase hosting
- Heroku
- react bootstrap
- react tostify
- env variable
- etc
